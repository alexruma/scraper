Something not working?

Please ensure that your roster file is formatted correctly:
  • Make sure that all providers you want to lookup are on the same sheet in your Excel workbook.
  • Make sure the worksheet that you want to search from is named "Sheet1" -the program will not run properly if the worksheet is named anything other than "Sheet1"
  • Make sure the provider's full name (format: Last, First) is in column F.
  • Make sure the provider's type is in column G.
  • Make sure the provider's specialty is in column I.
  • Make sure the provider's license number is in column K.
